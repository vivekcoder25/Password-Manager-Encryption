# 📖 Project Overview

The **Password Manager** is a Python application that securely stores and manages user passwords. Every password is encrypted using the **Fernet Encryption Algorithm** before being saved, ensuring that sensitive information remains protected.

This project demonstrates the use of Python programming, encryption, JSON file handling, password generation, and clipboard operations.

---

# ✨ Features

* 🔐 Master Password Login
* 🔒 Fernet Password Encryption
* ➕ Add New Password
* 👀 View Saved Passwords
* 🔍 Search Password
* 🗑 Delete Password
* 🎲 Generate Strong Password
* 📋 Copy Password to Clipboard
